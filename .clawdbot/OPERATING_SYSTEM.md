# OpenClaw Initiative Operating System

This document describes the persistent `.clawdbot` operating layer for PKRelay.

The goal is to make OpenClaw behave like a Chief of Staff agent with durable state, not just a chat session or spawn wrapper.

Use the shared `.clawdbot/scripts/*initiative*` commands to manage:

- active initiative state
- task graph
- artifacts
- reviews
- QA loop
- resumable session summaries
