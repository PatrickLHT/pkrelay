Getting Started

  1. Start the OpenClaw gateway: openclaw gateway restart
  2. Set your browser name above (e.g., "Arc")
  3. Click Test Connection to verify
  4. Click the PKRelay icon in the toolbar to manage tab permissions
  5. Attach tabs by clicking the green/gray dot next to each tab in the popup
